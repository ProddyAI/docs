## **Sending a Message in Chat**

To send a message:

1. Go to the **general channel** from the left sidebar.
2. Ensure the **Chat** tab is selected in the top bar.
3. Click on the message input field at the bottom where it says _“Ask a question about your workspace…”_.
4. Type your message.
5. Press **Enter** or click the **Send** button.

Your message will appear instantly in the chat.

## **Replying to a Message in a Thread**

To reply to a message in a thread:

1. Hover over the message you want to reply to.
2. Click on the **Reply** or **Thread** option (if visible).
3. Type your reply in the thread input box.
4. Press **Enter** to send the reply.

Thread replies help keep conversations organised.

## **Mentioning a User in Chat**

To mention a user:

1. Click on the message input field.
2. Type **@** followed by the user’s name.
3. Select the user from the list.
4. Type your message and send it.

Mentioned users receive a notification.

## **Adding a Message to Tasks or Calendar**

To convert a message into an action:

1. Hover over the message you want to convert.
2. Right-click on the message or click the message options menu.
3. Select the option to:
   - Add to **Tasks**, or
   - Add to **Calendar**
4. Confirm the action if prompted.

This links communication directly to execution.

## **Searching Messages in Chat**

To search messages:

1. Click on the **Search** bar at the top of the workspace.
2. Enter keywords related to the message you’re looking for.
3. Review the search results and click on a message to view it in context.

## **Pinning or Highlighting Important Messages**

To highlight important messages (if available):

1. Hover over the message you want to highlight.
2. Click on the message options menu.
3. Select **Pin** or equivalent option.

Pinned messages remain easily accessible.

1. Open a channel and navigate to the **Chat** tab.
2. Click on the **Calendar  icon** in the message input toolbar at the bottom.
3. The **Add Calendar Event** dialog box will appear.
4. Select a date using one of the available options:
   - **Today**
   - **Tomorrow**
   - **Next week**
   - **Custom** (to choose a specific date)
5. (Optional) Set a time for the event.
6. Review the event details.
7. Click on **Add to Calendar** to confirm.
